第一节：九针之名，各不同形

“九针之名，各不同形。一曰鑱针，长一寸六分。二曰员针，长一寸六分。三曰鍉针，长三寸半。四曰锋针，长一寸六分。五曰铍针，长四寸，广二分半。六曰员利针，长一寸六分。七曰毫针，长三寸六分。八曰长针，长七寸。九曰大针，长四寸。”

黄帝曰：夫九针为之也，各有所宜，以应九变。久风湿痹，筋挛骨痛，可用长针、大针。病在肌肤，可用员针、鍉针。病在脉，气少，不足以实，可用毫针补之。病在经络，可用锋针出血。总之，因病施治，针具选择至关重要。

“知其要者，一言而终，不知其要，流散无穷。” —— 倪海厦

倪师强调，学习中医，无论是针灸还是方剂，都必须抓住核心。九针之用，关键在于辨证。医生需明晰病位之深浅、病性之虚实，方能选用最合适的针具，达到“一针见效”的目的。例如，对于深层的风湿顽痹，非长针、大针不能及也；而对于浅表的皮肤病，若误用长针，则可能伤及正气。
