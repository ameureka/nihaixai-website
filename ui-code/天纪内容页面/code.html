名词索引

  • 天人合一

    中国哲学思想，指天与人、自然与人事的相通、相类和相应。

  • 八卦 (Bā Guà)

    《周易》中的八种基本图形，由三个爻组成，象征自然界的八种基本物质。

  • 五行 (Wǔ Xíng)

    指金、木、水、火、土五种物质的运动变化，用以说明世界万物的形成及其相互关系。
